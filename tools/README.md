# ✂ Вырезатель — pixel-perfect извлечение объектов и SFX

Маленький автономный инструмент (один `index.html`, без сборки и интернета) для
аккуратного выреза объектов с фона. Заточен под **SFX-ассеты** — огонь, дым,
молнии — где важна **полупрозрачность каждого пикселя**, а не грубая маска.

Открыть: просто дважды кликнуть `tools/index.html` в браузере. Всё считается
локально, картинки никуда не уходят.

## Как это устроено

Конвейер **неразрушающий** — оригинал не портится, любой шаг отменяется:

```
base (RGBA оригинал) ──► luma-key ──┐
                                     ├──► итоговая альфа = lumaA × mask/255
        ручная маска (палочка+кисть)─┘
```

## Авто-вырез (одна кнопка)

- **✨ Авто SFX (по яркости)** — сам подбирает пороги яркости методом **Отсу**
  для огня/дыма/молний на тёмном фоне. Мгновенно, оффлайн.
- **✨ Авто фон (от краёв)** — убирает ровный фон, связанный с краями кадра
  (берёт цвет рамки и допуск автоматически). Мгновенно, оффлайн.
- **🤖 ИИ-вырез (любое фото)** — нейросеть (`@imgly/background-removal`) вырезает
  объект с любого сложного фона. Первый запуск качает модель с CDN (нужен
  интернет), дальше работает из кэша. Для полупрозрачного дыма luma точнее.

После любого авто маску можно доточить кистью.

## Три режима (ручной контроль)

### 1. SFX / luma  (огонь · дым · молнии на тёмном фоне)
Альфа берётся не из «попал/не попал», а из **яркости** — поэтому глоу, искры и
полупрозрачные хвосты дыма сохраняются плавно, без «лесенки».

- **Канал** — `Макс RGB` (по умолчанию, ловит цветной огонь даже тусклый),
  яркость Rec.709 или отдельный R/G/B.
- **Ключ-цвет** — какой фон убираем (обычно чёрный; можно белый/любой).
- **Чёрная / белая точка** — диапазон яркости, который растягивается в 0…100% альфы.
- **Гамма** — форма перехода (тоньше/толще хвосты).
- **Восстановить цвет (unmultiply)** — поднимает тусклый цвет до истинного оттенка
  для обычного альфа-композитинга. Оставь **выкл.**, если кладёшь эффект
  аддитивно / режимом «Screen».

### 2. Волшебная палочка  (сплошной / ровный фон)
Клик по фону — убирается похожий по цвету участок.
- **Допуск** — порог похожести цвета.
- **Мягкий край** — анти-алиас на границе (не рвёт пиксели).
- **Смежные / Везде** — связная область или весь похожий цвет на картинке.

### 3. Кисть  (произвольное фото, ручная доводка)
- **Стереть / Вернуть** — убрать в прозрачность или восстановить исходник.
- **Размер · Жёсткость · Нажим**.
- <kbd>Alt</kbd> — временно противоположное действие.

## Pixel-perfect просмотр
- Зум nearest-neighbor — при увеличении **виден каждый пиксель**, без сглаживания.
- Шахматка / чёрный / белый фон для контроля прозрачности.
- В подвале — координаты пикселя, его RGB и итоговая **α** под курсором,
  и доля прозрачных пикселей.

## Горячие клавиши
| Клавиша | Действие |
|---|---|
| `1` `2` `3` | режим luma / палочка / кисть |
| колесо | зум к курсору |
| <kbd>Shift</kbd>+колесо | размер кисти |
| <kbd>Space</kbd>/средняя кнопка + тащить | панорама |
| <kbd>Alt</kbd> (с кистью) | инвертировать действие |
| <kbd>Ctrl/⌘</kbd>+<kbd>Z</kbd> / <kbd>Shift</kbd>+<kbd>Z</kbd> | отмена / повтор |
| <kbd>Ctrl</kbd>+<kbd>V</kbd> | вставить картинку из буфера |
