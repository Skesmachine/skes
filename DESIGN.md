# Design

## Visual Theme

Комикс-КЦГ-максимализм («бустер энергетиков»): светлый тёплый стол, на котором лежат кричащие коллекционные карты. Рисованная графика — чернильные контуры, комикс-бёрсты, поп-арт точки, молнии и звёзды (всё SVG/CSS, никаких фототекстур). 3D-банки в стиле PS1: низкополигональные, пиксельная флэт-текстура, тун-шейдинг. Референсы: Liquid Death, Duolingo, упаковки энергетиков, Pokémon TCG / Hearthstone, винтажные игровые автоматы.

## Color Palette

- `--ink: #1a140e` — чернила: контуры, текст, жёсткие тени. Все границы 2.5–4px solid ink.
- `--bg: #f6ead0` — тёплый стол (фон); поверх — дудлы и хафтон-точки.
- `--s1: #fffaf0` — кремовая поверхность карточек и кнопок.
- Акцент-огонь: градиент `#ffd56b → #ff8a2a` (primary-кнопки, лого).
- Тир-цвета (редкость): S `#dba008` золото+голо, A `#e2641a`, B `#d23c3c`, C `#5f9e22`, D `#8a8478`.
- Неон-акценты точечно: кислотный `#39ff14` (лассо-кроп), для стикеров — чистые жёлтый/красный.

Стратегия: Full palette — тир-цвета несут смысл (редкость), огонь — действие, чернила — структуру.

## Typography

Тройка шрифтов: **Rubik** — body; **Unbounded** — крупные заголовки (имена, формы); **Press Start 2P** (пиксельный, аркадный) — ЛОГОТИП «Скес Машина» (без рамки/градиента: оранжевый #ff8a2a + комиксная чернильная обводка text-shadow), заголовки страниц, вкладки, тумблеры, оценки, счётчики, стикеры, буквы тиров. Пользователю пиксельный шрифт особенно нравится — расширять в его сторону. Заголовки страниц компактные на мазке маркера (НЕ комикс-взрыв — пользователь отверг), главный акцент шапки — логотип с glow-молнией. Body ≥ 4.5:1 контраст.

## Components

- **.rcard** — коллекционная карта (Hearthstone-пропорции): бёрст за банкой по тиру, can-snap 3D-рендер, ленточка имени поверх арта, стикеры (НОВИНКА/ОГОНЬ!/ТРЕШ!/УЦЕНКА), ауры редкости (S голо-свип, A/B пульс, C кислота, D потёртая), разброс поворотов `--rot`, deal-in анимация раздачи.
- **.btn** — псевдо-3D: рукописность через НЕРОВНЫЕ углы (border-radius с разными значениями на каждый угол, напр. `12px 9px 13px 8px / 9px 13px 8px 12px`) + ровный 4px ink-бордер; офсет-тень 5px, hover приподнимается, active вдавливается. То же у .seg и .nav-tabs. ⚠ border-image-подход пробовали — растягивает углы криво, пользователь отверг.
- **.seg** — тумблер-переключатели, uppercase, inset-тень на активном.
- **.feed-title** — комикс-бёрст SVG за заголовком, кремовый текст с чернильной тенью.
- **Лассо-кроп** — канвас с неоновыми точками, контур, «Сбросить точки».
- **3D-банка** — SEG=20 flat shading, MeshToonMaterial 4 ступени, лейбл только на фронтальной дуге, PIX=6 (подбирался пользователем: 16 — мазня, 4 — слишком гладко), палитра 6 цветов, без аутлайна; бобинг + медленное вращение.
- **Логотип** — пиксельный «Скес Машина» 23px (мобайл 14px), оранжевый #ff8a2a БЕЗ обводки (ЛЮБАЯ обводка text-shadow заливает просветы PS2P → чёрное пятно; уже дважды наступали), только 2 слоя выдавленной грани. ⚠ text-shadow НАСЛЕДУЕТСЯ: у псевдоэлемента блика обязательно text-shadow:none, иначе он перекрашивает буквы в чёрный. Бобинг ±1px, блик по буквам на hover. 2D-молния 34×43 с двумя ступенями, глубиной; свечение — размытый эллипс ИЗ ЦЕНТРА внутри SVG (.bolt-glow, пульс opacity), не drop-shadow по контуру; линии скорости убраны (пользователь отверг). Бобинг банок мягкий (1px/−2px).
- **Обводка кнопок** — «тушь на бумаге»: border:4px transparent + двухслойный background (заливка padding-box, плитка --ink-stroke border-box) — тёмная тушь с просветами бумаги и царапинами, повторяет скругления. Применено к .btn/.seg/.nav-tabs.
- **Партиклы открытия карточки** — partyHTML(tc,letter): залп 16 частиц (звёзды/молнии/кружки/плюсы, размер 11–28px, полёт 1.1–1.8s, гаснут только в конце) + 3 плавающих стикера (искра, молния, слово по тиру POW_WORDS). ВСЁ рандомное на каждое открытие (Math.random, позиции стикеров инлайном); верхний правый угол не занимать — там крестик.
- **Блик карточки** — отдельный элемент .rcard-shine (не ::after — тот занят поверхностью карты): градиент ездит background-position'ом ВНУТРИ элемента, за края не вылезает (translateX-подход давал «припаркованное» пятно сбоку — не повторять).
- PIX подбирался: 16 мазня → 4 гладко → 6 → **5 финал**.
- **.rcard-back** — картонная подложка: inset ≈ −9..−12px (меньше не видна!), тёмный картон #cdb183→#9c8052, поворот ±1.2deg, мягкая тень.

## Layout

Лента — сетка маленьких карт с лёгким разбросом (повороты, наслоения ленточек на арт). Экспанд обзора — отдельная страница, банка выпирает за рамку арта на ~10%. Мобайл — нижняя навигация, тосты выше неё. Асимметрия и наклоны приветствуются; ровная «магазинная» сетка — нет.

## Worn & Torn (Hearthstone-потёртость)

Рваные края — фирменный приём, но **естественные**: мелкая неровная кромка обрыва бумаги (много мелких случайных зубцов), НЕ равномерная «ёлочка» и НЕ чернильная обводка по краю — мягкая drop-shadow тень. Под шапкой и над нижней навигацией — кремовый лист + картон из-под него (SVG repeat-x 480px). У каждой карты (.rcard-back) — рваная картонная подложка clip-path'ом, выглядывает на 2–3px. ВАЖНО: поверхность карты дублируется в .rcard::after (z-index:-1, после .rcard-back в DOM) — минусовые z-дети рисуются ПОВЕРХ фона элемента, без этого картон просвечивает сквозь карту. Никаких блик-свипов на обычных картах (голо-перелив только у S-тира). Никаких фототекстур.

## Бренд и вкус

drink_name хранится одной строкой «Бренд ¦ Вкус» (разделитель BF_SEP='¦', без миграции БД). Хелперы brandOf/flavorOf/dispName/brandKey. На карте: лента = бренд, под ней строка вкуса. В форме два поля с просьбой писать «точь-в-точь как на банке». Кнопка «Бренды» в ленте → модалка-список (группировка normDrink по бренду) → фильтр карточек.

## Motion

Ease-out (quart/expo), без bounce. Deal-in раздача карт со стаггером 60ms; бобинг банок sin-волной; голо-свип на S-картах; hover на карте → живое 3D-вращение банки. Переключение вкладок — скользящий «ползунок» (.slide-pill, JS refreshPills + MutationObserver), страницы появляются мягким fade+slide (page-in 0.3s). `prefers-reduced-motion: reduce` отключает бобинг, ауры, deal-in, page-in и ползунок.
