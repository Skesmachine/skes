# Design

## Visual Theme

Комикс-КЦГ-максимализм + **высеченный камень** (с 13.06.2026): на светлом тёплом столе лежат коллекционные карты из тиснёного камня (Hearthstone-рамка). Каменная плита с пиксельной брусчаткой-фактурой (присланная пользователем PNG-текстура подземелья, ужата до тайла 32×32, `image-rendering:pixelated`), круглое высеченное окно-ниша под банку, пергаментные плашки текста, гекс-/чернильные контуры. 3D-банки в стиле PS1: низкополигональные, пиксельная флэт-текстура, тун-шейдинг. Референсы: Liquid Death, Duolingo, упаковки энергетиков, Pokémon TCG / Hearthstone, пиксель-арт подземелий, винтажные игровые автоматы.

## Color Palette

- `--ink: #1a140e` — чернила: контуры, текст, жёсткие тени. Все границы 2.5–4px solid ink.
- `--bg: #f6ead0` — тёплый стол (фон); поверх — дудлы и хафтон-точки.
- `--s1: #fffaf0` — кремовая поверхность карточек и кнопок.
- Акцент-огонь: градиент `#ffd56b → #ff8a2a` (primary-кнопки, лого).
- Тир-цвета (лестница ККИ): S `#dba008` золото, A `#9347d6` фиолетовый, B `#2e7fd0` синий, C `#5f9e22` зелёный, D `#8a8478` серый.
- Неон-акценты точечно: кислотный `#39ff14` (лассо-кроп), для стикеров — чистые жёлтый/красный.

Стратегия: Full palette — тир-цвета несут смысл (редкость), огонь — действие, чернила — структуру.

## Typography

Тройка шрифтов: **Rubik** — body; **Unbounded** — крупные заголовки (имена, формы); **Press Start 2P** (пиксельный, аркадный) — ЛОГОТИП «Скес Машина» (без рамки/градиента: оранжевый #ff8a2a + комиксная чернильная обводка text-shadow), заголовки страниц, вкладки, тумблеры, оценки, счётчики, стикеры, буквы тиров. Пользователю пиксельный шрифт особенно нравится — расширять в его сторону. Заголовки страниц компактные на мазке маркера (НЕ комикс-взрыв — пользователь отверг), главный акцент шапки — логотип с glow-молнией. Body ≥ 4.5:1 контраст.

## Components

- **.rcard** — каменная коллекционная карта: плита из тиснёного камня (`--stone-cobble` overlay + градиенты + направленная фаска через box-shadow для объёма), круглое высеченное окно-ниша `.rcard-niche` (тёмная утопленная чаша с каменным бортиком), за банкой бёрст в цвет тира (обрезан по кругу), `can-snap` 3D-рендер выпирает вверх из ниши, инфо-зона — пергаментная плашка с тёмным текстом, подвал — утопленная каменная полка. Единственная наклейка — НОВИНКА. У каждой карты свой участок породы (сдвиг `--gx/--gy` по сиду id). Разброс поворотов `--rot`, deal-in раздача. На наведении банка увеличивается (`scale` независимым свойством, чтобы не конфликтовать с бобингом-`translate`).
- **.btn** — псевдо-3D: рукописность через НЕРОВНЫЕ углы (border-radius с разными значениями на каждый угол, напр. `12px 9px 13px 8px / 9px 13px 8px 12px`) + ровный 4px ink-бордер; офсет-тень 5px, hover приподнимается, active вдавливается. То же у .seg и .nav-tabs. ⚠ border-image-подход пробовали — растягивает углы криво, пользователь отверг.
- **.seg** — тумблер-переключатели, uppercase, inset-тень на активном.
- **.feed-title** — комикс-бёрст SVG за заголовком, кремовый текст с чернильной тенью.
- **Лассо-кроп** — канвас с неоновыми точками, контур, «Сбросить точки».
- **3D-банка** — SEG=20 flat shading, MeshToonMaterial 4 ступени, лейбл только на фронтальной дуге, PIX=6 (подбирался пользователем: 16 — мазня, 4 — слишком гладко), палитра 6 цветов, без аутлайна; бобинг + медленное вращение.
- **Логотип** — пиксельный «Скес Машина» 23px (мобайл 14px), оранжевый #ff8a2a БЕЗ обводки (ЛЮБАЯ обводка text-shadow заливает просветы PS2P → чёрное пятно; уже дважды наступали), только 2 слоя выдавленной грани. ⚠ text-shadow НАСЛЕДУЕТСЯ: у псевдоэлемента блика обязательно text-shadow:none, иначе он перекрашивает буквы в чёрный. Бобинг ±1px, блик по буквам на hover. 2D-молния 34×43 с двумя ступенями, глубиной; свечение — размытый эллипс ИЗ ЦЕНТРА внутри SVG (.bolt-glow, пульс opacity), не drop-shadow по контуру; линии скорости убраны (пользователь отверг). Бобинг банок мягкий (1px/−2px).
- **Обводка кнопок** — «тушь на бумаге»: border:4px transparent + двухслойный background (заливка padding-box, плитка --ink-stroke border-box) — тёмная тушь с просветами бумаги и царапинами, повторяет скругления. Применено к .btn/.seg/.nav-tabs.
- **Партиклы открытия карточки** — partyHTML(tc,letter): световой блум (`screen`) + цветная вспышка + двойная ударная волна + залп крупных осколков (16–38px, st/bo/ci/tr) + фонтан светящихся искр (`.lp`, screen, цвет тира + тёплый белый) + 2 плавающих стикера (звёздочка и молния — силуэт через CSS-`mask` + `background:var(--tc)`, в цвет тира). Слов («ВЖУХ!/НЕПЛОХО!/ХЛАМ!») НЕТ — пользователь отверг. Кол-во по тиру D5/C7/B9/A12/S15; верхний правый угол не занимать — там крестик.
- **Блик карточки** — отдельный элемент .rcard-shine (не ::after — тот занят поверхностью карты): градиент ездит background-position'ом ВНУТРИ элемента, за края не вылезает (translateX-подход давал «припаркованное» пятно сбоку — не повторять).
- PIX подбирался: 16 мазня → 4 гладко → 6 → **5 финал**.
- **.rcard-back** — картонная подложка: inset ≈ −9..−12px (меньше не видна!), тёмный картон #cdb183→#9c8052, поворот ±1.2deg, мягкая тень.
- **Текстура камня** — `--stone-cobble`: присланный пользователем PNG подземелья, ужат до тайла 32×32 (степень 10) с поднятой яркостью (+18%), квантован (~2.5 КБ base64), наложен `background-blend-mode:overlay` поверх каменных градиентов. `image-rendering:pixelated` держит пиксели чёткими; банка и 3D возвращены к `auto`. Чтобы заменить текстуру/степень — пересоздать тайл и `--stone-cobble`.

## Layout

Лента — сетка маленьких карт с лёгким разбросом (повороты, наслоения ленточек на арт). Экспанд обзора — отдельная страница, банка выпирает за рамку арта на ~10%. Мобайл — нижняя навигация, тосты выше неё. Асимметрия и наклоны приветствуются; ровная «магазинная» сетка — нет.

## Worn & Torn (Hearthstone-потёртость)

Рваные края — фирменный приём, но **естественные**: мелкая неровная кромка обрыва бумаги (много мелких случайных зубцов), НЕ равномерная «ёлочка» и НЕ чернильная обводка по краю — мягкая drop-shadow тень. Под шапкой и над нижней навигацией — кремовый лист + картон из-под него (SVG repeat-x 480px). У каждой карты (.rcard-back) — рваная картонная подложка clip-path'ом, выглядывает на 2–3px. ВАЖНО: поверхность карты дублируется в .rcard::after (z-index:-1, после .rcard-back в DOM) — минусовые z-дети рисуются ПОВЕРХ фона элемента, без этого картон просвечивает сквозь карту. Никаких блик-свипов на обычных картах (голо-перелив только у S-тира). Карта теперь каменная (`.rcard-back` картон сохранён под плитой); фактура камня — единственная допустимая «фототекстура» (пиксельная, overlay).

## Система оценок (с 12.06.2026)

5 мономодальных шкал 0–10 с весами (Σ=100): Сексуальность 30, Покурность 25, Рыгучесть 15, Благородность 15, Депрессивность 15. Оценка = взвешенная сумма /100 → 0..10. Тумблеры Тахикардия и Тремор: каждый +10% к итогу (максимум 12/10). Оценка >10 переливается радугой (rainbow-cycle) и крупнее. Хранение БЕЗ миграций БД: шкалы в старых колонках p1..p4 + tahikardia; маска тумблеров в tremor. Новый формат = 200+v (отличимо от старых: <100 — биполярки центр-5, 100..199 — биполярки 100+v20); декодеры monoVal()/togglesOf() понимают все поколения. Редкость: Обычный/Необычный/Редкий/Эпический/Легендарный (= D/C/B/A/S), LEGACY_TIER_MAP маппит старые вердикты («Проходняк» и т.п.).

## Бренд и вкус

drink_name хранится одной строкой «Бренд ¦ Вкус» (разделитель BF_SEP='¦', без миграции БД). Хелперы brandOf/flavorOf/dispName/brandKey. На карте: лента = бренд, под ней строка вкуса. В форме два поля с просьбой писать «точь-в-точь как на банке». Кнопка «Бренды» в ленте → модалка-список (группировка normDrink по бренду) → фильтр карточек.

## Редкость = главный визуальный сигнал

Тиры: Полная хуйня (D) / Хуйня (C) / Норм хуйня (B) / Заебись хуйня (A) / Легендарная хуйня (S). Редкость НЕ выбирается — вычисляется из оценки (suggestTier/tierOfReview): 1-2/3-4/5-6/7-8/9-12. Поля выбора в форме нет.

**Разная порода камня + «хром» по тиру** (цвет тира `--tc` задаёт JS; оттенок породы `--frame` — в `.rcard.l*` и `FRAME_BY_LETTER` для разворота). Светлый камень: D `#8a8175` (выветренный серый, `saturate .82`), C `#717a52` (замшелый), B `#687589` (синеватый гранит), A `#786a86` (фиолетовый сланец + holo-фольга `.rcard.lA::before`), S `#9c8350` (золотой камень + круглый силуэт 30px + полный голо-перелив `.tier-glow::before` + мерцающая искра). Свечение `.rcard-glow` (z:-2, за подложкой, blur 18px): B 0.30 / A 0.42 дышит / S 0.55 быстро. Бёрст `.burst` вращается (burst-spin) от A (24s) и S (14s). Трещины-молнии пробовали — пользователь отверг, убраны. Перекалибровка тиров — `--frame` + `--glow`.

## Motion

Фирменные кривые в :root: --ease-out (0.22,1,0.36,1), --ease-fly, --ease-glide. НИКАКОГО bounce и scale-from-tiny. Блик карты (.rcard-shine): transition только на :hover → при уходе мыши не едет обратно, мгновенно на парковку. Голо-свип S (holo-sweep) зациклен бесшовно: полоса полностью уходит за край + пауза до рестарта (скачок невидим). Бобинг банок sin-волной; hover на карте → живое 3D-вращение; вкладки — скользящий .slide-pill (refreshPills + MutationObserver); страницы — fade+slide page-in.

**Анимация открытия карточки v2 (комикс-удар, масштаб по тиру):** `.exp-flash` — радиальная вспышка-полутон в цвет тира (≈0.55s); `.exp-shock`×2 — двойное комикс-кольцо с чернильной обводкой, резкий выброс + длинный выбег, радиус `--shock-max` (D×5…S×11); `.pp` — КРУПНЫЕ осколки (16–38px, формы st/bo/ci/tr), цвет из палитры тира, у каждого своя скорость (--pp-dur 1.4–2.2s), держатся в кадре и уходят сносом вверх (--dx2/--dy2). Кол-во по тиру D5/C7/B9/A12/S15. Контейнер `.exp-party{overflow:hidden}` срезает осколки по рамке арта. Микро-жизнь: пульс сердечка-лайка, вигл молнии-лого, подскок иконок вкладок, крен аватаров/плиток, качание банки в empty-state.

**Открытие быстрое и плавное:** тяжёлая инициализация 3D-сцены (WebGL) отложена на 360мс — не конкурирует с анимацией открытия (фото видно сразу, 3D подхватывается следом). FLIP 0.34s, выезд блоков `.expand-reveal` 0.34s. Hover-3D в ленте — задержка 90мс + банка сразу увеличивается (`scale`), 3D подхватывает то же увеличение без скачка.

`prefers-reduced-motion: reduce` отключает бобинг, deal-in, page-in, ползунок, .pp/.exp-shock/.exp-flash/.exp-bloom/.lp, holo-фольгу, burst-spin, micro-hover.
